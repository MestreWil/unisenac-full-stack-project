## What is MusicRATE?
This project aims to provide a web platform where people can share their opinions about albums or artists, based on reviews and scores ranging from 0 to 5. Our idea is inspired by the IMDb website and our passion for listening to music. We are aware that there may be some bugs, so please feel free to explore the code and share any suggestions for improvements. ðŸ˜‰

## Tools
* Django REST framework for backend: [Django-REST-framework](https://www.django-rest-framework.org/)
* React with NextJs framework for frontend: [Nextjs](https://nextjs.org/)
* PostgreSQL for Database: [PostgreSQL](https://www.postgresql.org/)
* Postman to test API: [Postman](https://www.postman.com/)
